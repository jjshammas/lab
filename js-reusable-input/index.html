Enhance access to commonly-used text fields

This lightweight jQuery plugin makes it easy to overwrite current text in an input without a double-click or select-all. However, the user can still regain normal functionality and edit the previously entered text if necessary.

Default implementation

$("#search-field").reusableInput();

Click and the caret is set to the beginning of the input. Click again and default functionality is resumed.

Disable/modify blur

$("#search-field").reusableInput({
	blur: false
});

Caret will still be moved, input will not change color

$("#search-field").reusableInput({
	blurColor: "#FA715F"
});

Custom caret position

$("#search-field").reusableInput({
	caretPosition: 4,
	blur: false
});

Useful for navigating through directories, URLs, etc.

One time use

$("#search-field").reusableInput({
	once: true
});

After the input is blurred, default functionality will take over permanently.
